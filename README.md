# RestWithASP.NET5

## Scaffonding command

### Create a solution :
```bash
dotnet new sln -o RestWithASPNET
```
### Go to the directory
```bash
cd .\RestWithASPNET\
```
### Create a WebAPI project
```bash
dotnet new webapi -o RestWithASPNET
```
### Associate Solution to the Project
```bash
dotnet sln add .\RestWithASPNET\RestWithASPNET.csproj
```
### Run app with certificate
```bash
dotnet run
```
